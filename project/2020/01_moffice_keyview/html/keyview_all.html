당일 해제키 입력 방법
아래 2가지 방법 중 각 사에 해당하는 방법으로 당일 해제키를 입력하시면 차단창이 해제되어 PC 사용이 가능합니다.
  • 차단창 하단 종료 버튼 클릭 > 우측 메뉴에서 당일 해제키 클릭 > 당일 해제키 입력 > 차단창 해제
  • 차단창 우측 상단 로고 클릭 > 당일 해제키 입력 > 차단창 해제
당일 삭제키 입력 방법
각 PC의 제어판 클릭 > 프로그램 추가/제거 클릭 > 삭제 값 부분에 당일삭제키 입력 > 삭제 확인
