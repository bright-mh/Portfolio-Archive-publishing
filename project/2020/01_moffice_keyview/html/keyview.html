차단창 해제 방법

PC 차단창 또는 자리비움 차단창에 해당하는 PC 사용키를 입력하시면 차단창이 해제되어 PC사용이 가능합니다.

  • 1. PC차단창

    우측 하단의 [종료] 버튼 클릭 > [PC사용키 입력] 버튼 클릭 > PC 사용키 입력

  • 2. 자리비움 차단창

    우측 하단의 [자리비움 종료] 버튼 클릭 > [PC사용키 입력] 버튼 클릭 > PC 사용키 입력

프로그램 삭제 방법

사용자에게 전달받은 인증키 6자리를 입력 후 생성된 삭제값을 입력하거나 당일 삭제키를 입력하시면 프로그램 삭제가 가능합니다.

  • 1. 해당 PC만 삭제 가능

    제어판 클릭 > 프로그램추가/제거 클릭 > 삭제값 부분에 인증키 입력 후 생성된 삭제값 입력

생성

  • 2. 모든 PC에서 삭제 가능

    제어판 클릭 > 프로그램추가/제거 클릭 > 삭제값 부분에 당일 삭제값 입력
